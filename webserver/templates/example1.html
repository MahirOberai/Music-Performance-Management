Query Example 1

This query example displays the occupancies of the venues and runs the follow query:

SELECT v.name as venue_name, count(s.seat_id) as occupancy FROM seat s JOIN venue v ON v.venue_id = s.venue_id GROUP BY v.name ;

Output:
venue_name, occupancy /100 total occupancy
