Query Example 2

This query example lists artists by genre and then allows the user create a genre-specic show by placing the chosen artists into a venue.

The example runs the following query:

SELECT a.name as artist_name, a.genre as artist_genre, a.venue_id as venue_id FROM artist a ORDER BY artist_genre ;

Output:
